    DBeaver

    开源！免费

    密码存储位置：C:\Users\xxxx\AppData\Roaming\DBeaverData\workspace6\General.dbeaver
    默认存储密码的文件：credentials-config.json

    使用openssl对Json文件进行解密

    FileZilla

    密码位置：C:\Users\你的用户名\AppData\Local\finalshell\conn
    Json文件夹里复制里面的password

    Java解密

    HeidiSQL

    注册表位置：计算机\HKEY_CURRENT_USER\SOFTWARE\HeidiSQL\Servers\xxxx
    字段位置 ：LogFilePath=C:\Users\guagua\AppData\Roaming\HeidiSQL\Logs%session%db%y%m%d.sql
    PS：Servers目录下以登录名记录登录信息

    配置保存在：portable_settings.txt

    PS：[SSH_Hostkeys]带有ed25519字样的密码咱不能解密

    【安装有该软件】

    Navicat

    Navicat_11&12&15

    1、Navicat软件界面，导出连接
    2、注册表获取
    reg query HKEY_CURRENT_USER\SOFTWARE\PremiumSoft\Navicat\Servers
    reg query HKEY_CURRENT_USER\SOFTWARE\PremiumSoft\Navicat\Servers /s /v host 数据库连接 ip
    reg query HKEY_CURRENT_USER\SOFTWARE\PremiumSoft\Navicat\Servers /s /v UserName
    reg query HKEY_CURRENT_USER\SOFTWARE\PremiumSoft\Navicat\Servers /s /v pwd 数据库用户名数据库用户名对应的密码 hash
    reg query HKEY_CURRENT_USER\SOFTWARE\PremiumSoft\Navicat\Servers /s /v Port 据库连接

    向日葵

    使用小于V 12.1.0.48900（2022.12）

    安装版：C:\Program Files\Oray\SunLogin\SunloginClient\config.ini

    便携版：C:\Program\Data\Oray\SunloginClient\config.ini

    注意高版本的向日葵配置是放在注册表中
    reg query HKEY_USERS.DEFAULT\Software\Oray\SunLogin\SunloginClient\SunloginInfo
    reg query HKEY_USERS.DEFAULT\Software\Oray\SunLogin\SunloginClient\SunloginGreenInfo

    ToDeskTop

    安装目录下config.ini文件，字段tempAuthPassEx，复制出来然后替换本机的，能看到明文

    PhpAdmin

    【1】找到配置文件my.ini，文件中加入skip_grant_tables，然后重启wampserver

    【2】进入到数据库安装目录的bin文件夹下，cmd运行mysql，mysql -uroot -p

    【3】修改密码：update mysql.user set authentication_string=password(‘root’) where user=‘root’

    【4】重启服务然后删除ini文件中skip_grant_tables
