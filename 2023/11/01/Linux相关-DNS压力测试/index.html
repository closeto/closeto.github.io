    DNS服务器进行压力测试推荐工具dnsperf

    Linux压力测试参考链接：https://blog.51cto.com/topic/dns-linux-ceshigongju.html

    安装

    PS：不支持随机源地址

    ​ apt install dnsperf

    dnsperf参数介绍：

    使用

    ​ dnsperf -s dns服务器地址 -d dns记录列表文件
    ​ dnfperf -s 1.1.1.1 -f record.txt

    ⭐注意：-d 参数

    ​ 数据文件中以“#”开头的行被认为是注释行，会被dnsperf忽略。

    ​ 👇👇👇👇👇其中有效数据由两列组成，第一列是查询域名，第二列是查询的资源类型，dnsperf支持的资源类型如下👇👇👇👇👇：
    A，NS，MD，MF，CNAME，SOA，MB，MG，MR，NULL，WKS，PTR，HINFO，MINFO，MX，TXT，AAAA，SRV，NAPTR，A6，ASFR，MAILB，MAILA，ANY

    -d参数数据文件示例：domain.txt

    使用示例：

    ​ 使用hping3对DNS的 TCP-53 和 UDP-53进行Flood攻击

    ​ UDP：hping3 –flood -q -2 -E domain.txt dnsserver
    ​ TCP： hping3 –flood -q -E domain.txt dnsserver
